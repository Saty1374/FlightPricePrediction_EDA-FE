## 📌 Project Overview

This project focuses on analyzing flight data and building a **Machine Learning model** to predict flight ticket prices based on various factors.

The objective is to:
- 📊 Perform Exploratory Data Analysis (EDA)
- 🔍 Identify key factors affecting flight prices
- 🤖 Build a predictive model
- 📈 Improve accuracy using feature engineering

---

## 📂 Dataset Information

The dataset contains information about flight journeys including:

- Airline  
- Date of Journey  
- Source & Destination  
- Route  
- Departure & Arrival Time  
- Duration  
- Total Stops  
- Additional Information  
- Price (Target Variable)  

---

## 🔑 Key Features Explained

- **Airline** → Company operating the flight  
- **Date_of_Journey** → Travel date  
- **Source** → Starting location  
- **Destination** → Ending location  
- **Route** → Flight path  
- **Dep_Time** → Departure time  
- **Arrival_Time** → Arrival time  
- **Duration** → Total travel time  
- **Total_Stops** → Number of stops  
- **Additional_Info** → Extra details  
- **Price (Target)** → Ticket price

## 📊 Project Workflow

### 🔹 1. Data Cleaning
- Handled missing values  
- Removed inconsistencies  
- Converted categorical data  

### 🔹 2. Feature Engineering
- Extracted day, month from date  
- Converted duration into hours/minutes  
- Encoded categorical variables  

### 🔹 3. Exploratory Data Analysis (EDA)
- Distribution of prices  
- Airline vs Price analysis  
- Stops vs Price  
- Time-based trends  

### 🔹 4. Model Building
- Applied regression models:
  - Linear Regression  
  - Random Forest  
- Compared model performance  

---

## 📈 Key Insights

- ✈️ Flights with more stops tend to have lower prices  
- 🕒 Travel time significantly affects ticket price  
- 🏢 Airline plays a major role in pricing  
- 📅 Seasonal trends impact pricing
